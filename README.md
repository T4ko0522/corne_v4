## ハード仕様

| 項目 | 値 |
|---|---|
| ボード | Corne v4.1 (rev4_1/standard, RP2040) |
| USB VID/PID | `0x4653` / `0x0004` |
| 配列 | JIS |
| LED | RGB Matrix |
| 接続 | TRS (3極) split |
| エンコーダー | 右側 1個 |
